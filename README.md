## ✏️ 프로젝트 개요
저희 캡스톤디자인 프로젝트는 조선대학교 교내 셔틀버스의 실시간 위치를 제공하는 PWA(Progressive Web App)를 개발하는 것입니다. 이 프로젝트는 교내 셔틀버스 이용자들에게 실시간 셔틀버스 위치, 셔틀버스 내 분실물, 셔틀버스 이용 수칙 등의 교내버스 이용 관련 편리성을 제공하고자 기획되었습니다.
